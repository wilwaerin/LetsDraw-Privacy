# Lets Draw – Privacy Policy

## 🇬🇧 English

**Lets Draw** is a completely offline, single-player drawing application.

We do not collect, store, process, or share any personal data or user information.

The application does not require an internet connection and does not use advertisements, analytics, or any third-party tracking services.

No user accounts, login systems, or personal identifiers are used in this application.

All drawings created within the app are stored locally on the user's device. The app may save images to the device’s storage when the user chooses to export or save their drawings.

The application does not transmit any data outside of the user's device.
